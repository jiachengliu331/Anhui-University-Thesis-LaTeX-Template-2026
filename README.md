# AHUThesis2026

安徽大学（Anhui University）2026 年最新版硕士学位论文 LaTeX 模板。

## 项目定位

本项目基于公开论文模板进行适配与重构，面向安徽大学硕士学位论文写作场景。

当前版本重点强调：

1. 2026 年适配版本（Latest 2026 Edition）。
2. 核心改动集中在 [buaa.cls](buaa.cls) 的版式与结构控制。
3. 保留常见论文章节组织方式，便于直接替换内容并开始写作。

## 项目新名称

中文名：安徽大学 2026 硕士论文 LaTeX 模板  
英文名：Anhui University Thesis LaTeX Template 2026  
仓库名建议：AHUThesis2026

## 主要修改说明

与原始基础模板相比，本项目的主要工作在 [buaa.cls](buaa.cls)：

1. 调整论文版式相关参数与样式定义。
2. 适配安徽大学硕士论文常见格式需求。
3. 优化章节、封面与论文主体排版的一致性。

## 使用说明

1. 打开 [Template.tex](Template.tex) 作为主文件编译。
2. 在 [tex](tex) 目录下编辑各章节内容。
3. 参考 [ref.bib](ref.bib) 维护参考文献。
4. 按需选择 [bst](bst) 目录下的国标样式文件。

## 环境建议

推荐使用以下任一 LaTeX 工具链：

1. Overleaf + XeLaTeX
